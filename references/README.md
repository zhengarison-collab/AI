# Portfolio reference package

`reference-assets.zip` contains the source materials Terry supplied for the personal portfolio build.

## Contents

- `visual/homepage-hero-reference-01.webp` — dark editorial homepage composition with cybernetic floral portrait.
- `visual/homepage-hero-reference-02.webp` — alternate homepage composition and headline treatment.
- `visual/cyber-floral-portrait-reference.webp` — vertical character and visual-language reference.
- `motion/portfolio-motion-reference.mp4` — short interaction and motion reference.
- `resume/zheng-chao-resume-2024.pdf` — resume source for About and Resume content.

## For Codex

Unpack before implementation:

```bash
unzip references/reference-assets.zip -d references/reference-assets
```

Treat these files as design and content references. Do not publish phone numbers, email addresses, or other private resume fields on the public website unless the repository owner explicitly requests it.

The uploaded media has been web-optimized to keep the repository lightweight while preserving its usefulness as a visual reference.
